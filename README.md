# Regression-Model-based-on-Boston-Housing-Dataset
Analysis of the Regression Model based on Boston Housing Dataset
Here Group CSTWY.py is the code file of this project.
It contains the detailed steps including three kinds of regression models: linear, kernel bridge and lasso.
Group CSTWY.pdf is the detailed report of this project.
In the pdf, we describe the whole process detailly, including data processing, modeling and testing, as well as the conclusion.
